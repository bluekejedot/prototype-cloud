# Prototype Cloud Project

## 📌 Deskripsi Proyek
Project ini merupakan prototipe sederhana implementasi sistem berbasis cloud menggunakan Python.  
Tujuan utama dari project ini adalah untuk memahami alur dasar pengembangan aplikasi berbasis cloud, version control menggunakan Git, serta integrasi dengan GitHub sebagai repository cloud.

Project ini juga menjadi dasar pembelajaran konsep **Cloud Computing**, khususnya pada aspek:
- Version Control System (VCS)
- Deployment sederhana berbasis script
- Struktur project cloud-ready
